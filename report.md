# Report

## Business Use Case
This project focuses on drafting customer support email responses for an e-commerce company. This task is important because customer support requires a lot of repetitive writing.

## Model Choice
I used the Gemini API with the google-genai Python package and the gemini-3-flash-preview model. I chose this model because it is fast and easy to use.

## Baseline vs Final Design
The initial prompt was very simple and produced generic responses. After revising the prompt, the final version produced more empathetic and structured responses.

## Evaluation
I created a small evaluation set with five test cases, including normal, edge, and difficult cases. This helped me test the system consistently.

## Limitations
The model still requires human review in some situations, especially when the input is unclear or sensitive.

## Recommendation
I recommend using this system as a support tool, not a fully automated system.
